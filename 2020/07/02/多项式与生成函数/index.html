多项式与生成函数

多项式与生成函数

参考

总是忘记多项式板子。。。

这里推一下柿子方便记忆

一些规定

本文中的 $\times$\times皆为卷积，表示$(A\times B)_i=\sum_{j+k=i}A_j\times B_k$

未特殊说明的话都在$\mod 998244353$意义下

FFT,NTT,MTT

略

INV

多项式求逆

已知函数$f$,求函数$g$满足$f\times g\equiv 1\mod x^n$

考虑倍增

假设已经求出函数$h$满足$f\times h\equiv 1\mod x^{n/2}$

两边同时平方

至于为什么可以这么做，考虑函数$A\equiv 0\mod x^n$,那么$\forall i\in[0,n)$,$A_i=0$

所以$A^2$中$\forall i\in[0,2n),A^2_i=0$

两边同时乘以$f$

LN

多项式求$\ln$

已知函数$f(x)$,求函数$g(x)$满足$g(x)\equiv \ln f(x)\mod x^n$

链式求导

之后求积分

EXP

多项式求$\exp$

已知函数$f$,求函数$g$满足$g\equiv e^f\mod x^n$

泰勒展开

将$f(x)$在$x_0$处泰勒展开，即构造一个函数，使他在$x_0$处的任意阶导数等于$f(x)$相同

牛顿迭代

已知函数$f$,求函数$g$满足$f(g(x))\equiv 0 \mod x^n$

考虑倍增

假设已知函数$h$满足$f(h(x))\equiv 0\mod x^{n/2}$

将$f$在$h(x)$处泰勒展开

显然，$f(g(x))\equiv 0 \mod x^{n/2}$

所以，$g(x)\equiv h(x)\mod x^{n/2}$

所以$\forall i>1,(g(x)-h(x))^i\equiv 0\mod x^n$

所以$f(g(x))\equiv f(h(x))+f’(h(x))\times (g(x)-h(x))\equiv 0\mod x^n$

EXP

两边同时取$\ln$

设$G(g(x))=\ln(g(x))-f(x)\mod x^n$

套牛顿迭代

假设已经求出$G(h(x))\equiv 0\mod x^{n/2}$

$g(x)=h(x)-\frac{G(h(x))}{G’(h(x))}$

$h(x)$是自变量，$f(x)$与他无关，算常量

留坑待填
